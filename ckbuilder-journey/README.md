# CKBuilder Journey

A 3-month technical dev log for learning the Nervos CKB ecosystem from first principles.

## Learning Goals

- Understand CKB's mental model from the official Nervos documentation.
- Build a clear foundation in the Cell Model, capacity, live/dead cells, scripts, and transactions.
- Learn how lock scripts, type scripts, witnesses, cell dependencies, and CKB-VM fit together.
- Practice with developer tools such as OffCKB and CCC.
- Progress toward Rust scripting, Molecule serialization, and custom verification logic.
- Keep a consistent public record of technical learning, questions, experiments, and shipped outputs.

## Weekly Reporting Cadence

- One weekly report is submitted every Sunday.

## What Goes Where

- `reports/` contains weekly CKBuilder reports for submission.
- `notes/` contains daily or topic-based study notes from the Nervos CKB docs and related resources.
- `screenshots/` contains visual evidence of tooling, transactions, terminal output, demos, or experiments.
- `experiments/` contains small practical builds, scripts, commands, test projects, and verification exercises.
